Contexte

Pour cette activité nous avons commencé par l'analyse du site Nolark pour vérifier la conformité aux législations : RGPD, droit d'auteur du contenu multimédia, nom de domaine.

Après installation des outils 'habituels' (JDK Java, IDE Netbeans, Wampserver) nous avons installé le site web ainsi que sa base de données en local afin de corriger quelques dysfonctionnements dans le code HTML relevés par le validateur HTML du W3C.

Ensuite nous avons apporté des améliorations pour la page des CGU en appliquant des règles CSS, et nous avons ajouté une page avec un simulateur de test d'alcoolémie, en faisant un prototype avec Pencil, et en respectant la charte graphique du site. Nous avons fait les corrections nécessaires dans un script JavaScript pour interagir avec le nouveau formulaire.

Pour optimiser les résultats des moteurs de recherche (SEO) nous avons implémenté un fichier 'sitemap.xml' et corrigé le fichier 'robots.txt' pour signaler ce nouveau sitemap et pour autoriser les robots à scanner le site. Nous avons également ajouté des balises 'strong' pour signaler le contenu du site aux robots et implémenté l'utilisation de balises HTML sémantiques.

Pour finir, nous avons mis le site 'en ligne' sur une machine virtuelle serveur Linux.
